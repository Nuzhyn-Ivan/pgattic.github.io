Challenges

Hello, and welcome to my Coding Challenges page! This is where I put all of my Coding Challenges that I do.

The rules of my Challenges are:

  • The resulting code must be less than or equal to 1 Kilobyte of space (that's 1024 characters).
  • It may not reference other pieces of code. This includes external js files, frameworks, or binaries for the purpose of executing tasks. Media files such as images, audio files, and video files are allowed.
  • If the file generates an error while running but still runs, it isn't allowed. The code is tested on Google Chrome, and some coding shortcuts taken in these may not work in other browsers.

And here are my Challenge projects so far:

  • Typewriter (72 bytes): Type anything in the box, and you will hear the sound of a typewriter clicking.
  • CPS counter (203 bytes): Click anywhere in the page as many times as you can for 5 seconds, and your clicks per second will be shown.
  • Paint (510 bytes): Use your mouse to draw stuff on the screen! Press the F5 key to reload the page (as with any web page).

I will continue to shrink the code for these whenever I see a possibility to do so.
