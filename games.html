These are the games I've made:

Open Snake

This is a simple snake game that is planned to be highly customizable. I plan on implementing some sandbox features.

Controls:

  • Move: Arrow keys or WASD
  • Pause: E

Note: The game starts off paused so that you can configure the settings until you're ready.

Shoot 'Em!

This is a more complex game, in which you are a ball, trying to survive against the enemy invasion.

  • Move: Arrow Keys or WASD
  • Pause: E
  • Use Super Ability: Q
  • Shoot: Mouse

You start off with three lives and three Super Ability uses, so use them wisely!
