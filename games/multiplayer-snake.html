Multiplayer Snake

Update from Snake that features playing with up to 4 players sharing one keyboard!

How to play:

The game starts off with no players in game. Add (or drop) a player by pressing the number key that corresponds to that player number. For example, you can add or remove player 1 in the game by pressing "1" on the keyboard.

Controls:

  • Player 1 move: WASD
  • Player 2 move: TFGH
  • Player 3 move: IJKL
  • Player 4 move: Arrow keys
  • Pause game: Space bar or Enter

Keep in mind that this game does not support mobile devices. If you are on a mobile device right now, please switch to a computer to play this.
